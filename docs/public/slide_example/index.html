主要修改内容与示例 (四分栏布局)

化繁为简
我认为在之前的版本中，我们增加了太多可选择的内容。 相反，很多东西被反复用到，例如四分块，因为在组合的大屏幕上， 将字体调小，每四分之一的屏幕放置一份内容非常清晰。
自定义 LaTeX 环境

\newenvironment{quarterblock}[1]{%
  \begin{minipage}[t]{\linewidth}%
    {\scriptsize #1\par}% 标题
    \uestcline
    \tiny % 正文
  }{%
  \end{minipage}%
}
制作逻辑
四分块创建的逻辑是这样子的：首先使用 columns 将整个页面分为两列， 然后在每一列中使用模板自定义的 quarterblock 环境。
